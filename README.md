# 性偏好评分网站

一个基于Web的性偏好评估工具，严格按照文字版评分表设计，使用Pornhub黄黑配色风格。

## 功能特点

- 🎯 **简洁界面**：采用Pornhub黄黑配色，界面简洁美观
- 📊 **6分制评分**：严格按照文字版评分表的sss、ss、s、no、X、?评分体系
- 📋 **标准评估**：完全按照文字版评分表的内容进行评估
- 💾 **本地存储**：评分数据保存在浏览器本地存储中
- 📤 **结果导出**：支持导出HTML格式的评分结果
- 📱 **响应式设计**：支持手机和桌面设备

## 评分标准

- **sss - 非常喜欢/渴望**：我对此非常喜欢，这让我非常兴奋
- **ss - 喜欢/可尝试**：我对此持积极态度，愿意或喜欢
- **s - 无感/中立**：不反感也不热衷，但可以接受
- **no - 不喜欢**：我倾向于避免，这会降低我的兴致
- **X - 绝对拒绝/底线**：突破底线，绝对无法接受
- **? - 不了解/好奇**：我对此的感受暂时不了解，待尝试

## 快速开始

### 1. 启动服务器

#### 方法一：使用Python（推荐）
```bash
# 进入项目目录
cd Sexual_Preference

# 启动服务器（Python 3）
py -m http.server 8080

# 或者使用python命令
python -m http.server 8080
```

#### 方法二：使用Node.js
```bash
# 进入项目目录
cd Sexual_Preference

# 使用npx启动http-server
npx http-server -p 8080 -o
```

#### 方法三：使用PowerShell
```powershell
# 在PowerShell中运行
powershell -Command "cd Sexual_Preference; py -m http.server 8080"
```

### 2. 访问网站

服务器启动后，在浏览器中访问：
- **主页**: http://localhost:8080
- **直接访问**: http://localhost:8080/index.html

### 3. 停止服务器

在运行服务器的终端窗口中按 `Ctrl+C` 停止服务器

## 使用流程

1. **启动服务器**：按照上述方法启动本地服务器
2. **访问主页**：在浏览器中打开 http://localhost:8080
3. **开始评分**：点击"开始评分"按钮
4. **完成评估**：按照分类逐一完成所有项目的评分
5. **查看结果**：完成所有评分后查看结果表格
6. **导出结果**：可以导出HTML格式的评分结果
7. **停止服务器**：按 `Ctrl+C` 停止服务器

## 技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript
- **数据存储**：浏览器本地存储（localStorage）

- **无外部依赖**：纯静态网站，无需数据库

## 故障排除

### 服务器启动失败
- **Python未安装**：请安装Python 3.x
- **端口被占用**：尝试使用其他端口，如 `py -m http.server 8081`
- **权限问题**：以管理员身份运行终端

### 网站无法访问
- **检查服务器状态**：确认服务器正在运行
- **清除浏览器缓存**：按 `Ctrl+F5` 强制刷新
- **检查防火墙**：确保8080端口未被阻止

### 评分数据丢失
- **检查浏览器设置**：确保允许本地存储
- **清除浏览器数据**：避免意外清除localStorage

## 注意事项

- **隐私保护**：请在私密、放松的环境中完成评估
- **诚实填写**：请根据内心真实感受评分
- **数据安全**：所有数据仅保存在本地，不会上传到服务器
- **伴侣分享**：完成后可与伴侣分享，促进沟通理解
- **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Edge、Safari）
